# vue-todo-list
**To-Do list application** using Vuex store.
___
### Installation
- Clone the repository:
```bash
git clone https://github.com/kitaminka/vue-todo-list.git
```
- Install dependencies:
```bash
yarn install
```
- Start the program in development mode:
```bash
yarn serve
```
- Compile for production:
```
yarn build
```
